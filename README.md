## プロダクト名

きらきらおなまえできあがり！

## コンセプト


質問に答えるだけでキラキラネームをゲット！

## 対象ユーザ

特別な名前が欲しい人・大好きな人<br>
特徴的なキャラクターの名前を作りたい人

## 利用の流れ

１．いくつかの二択の質問に答えます<br>
２．キラキラネームが作られ、命名書が印刷されます

## 推しポイント

ゼロフロントエンド！！！
ソフトウェアのバックエンドとハードウェアのみを使用し、ユーザーが画面を操作することは一切ありません。
