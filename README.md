# P2HACKS2025 アピールシート

## プロダクト名

きらきらおなまえできあがり！

## コンセプト

質問に答えるだけでキラキラネームをゲット！

## 対象ユーザ

特別な名前が欲しい人・大好きな人<br>
特徴的な名前を作りたい人

## 利用の流れ

１．いくつかの二択の質問に答えます<br>
２．キラキラネームが作られ、命名書が印刷されます

## 推しポイント

ゼロフロントエンド！！！<br>
ハードウェアを操作して質問に答えると、自動的にプリンタから命名書が印刷されます。<br>
ソフトウェアのバックエンドとハードウェアのみを使用し、ユーザーが画面を操作することは一切ありません。<br>

### 開発における工夫した点

ハードウェアを使用することが決まっていたので、<br>
ハッカソン前に予め勉強会をしました。<br>
今回の開発では、チームマネジメントを工夫しました。<br>
プロダクトの構造や実装機能を早々に決めることで、<br>
タスクの洗い出しや、タスクの割り振りなどタスクの発見及び分担しました。<br>
作業量が多いタスクが発生したとき、アプローチを何度も検討しました。<br>
使用技術に全員の得意分野を含めることで開発速度の向上を図りました。<br>
